~~~~~~~~  BAMAZON  ~~~~~~~~

This app is an Amazon-like storefront. The app will take in orders from customers and removes stock from the store's inventory. 

![text](./images/Bamazon.gif)


Technologies used:
Node.js
Javascript
SQL
mySql Work bench
Vagrant
ubuntu 14.04 server

How to use:
1. Clone repo onto your local machine & open CLI of your choosing

2. Enter the cloned repos directory directory

3. Run in vagrant (Virtual Machine)

Install VAGRANT/Virtualbox

Once in the cloned repository

Run the command vagrant up upon completion run command vagrant ssh 

Navigate to the code directory by running cd /var/code/

Now run the command node BamazonCustomer.js to test the app.
